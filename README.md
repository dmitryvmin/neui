# NewEngen UI #

#### Start the app
  * `npm start`

#### Build for Production
  * `npm run build`

#### Build for Development
  * `npm run dev`

#### Run Type Checking (Flow)
  * `npm run flow`

#### Run tests (Jest)
  * `npm run jest` or `npm t`

#### Update snapshot tests (Jest)
  * `npm run snapshot-update`

#### Code Coverage (Jest)
  * `npm run cover`

#### E2E testing (Cypress)
  * `npm run cypress`

#### Storybook (Style Guide)
  * `npm run storybook`
  * `npm run storybook:build`
